트리(Tree)
트리 : Node(노드, 데이터 저장), Branch(가지, 브랜치)를 이용해서, 사이클이 이루지 않도록 구성된 데티어 구조 //조직도
1. Node : 트리에서 데이터를 저장하는 기본 요소 (데이터와 다른 연결된 노드에 대한 Branch  정보 포함)// 한칸 한칸
2. RootNode : 트리에서 맨 위의 시작 노드 // 제일 꼭대기
3. Level(depth): 최상위 노드를 level 0으로 하였을때, 하위 Branch로 연결된 노드의 깊이를 나타냄 //몇층짜리인지
4. Parent Node(부모노드): 어떤 노드의 위 Level 노드 //상급자
5. Child Node(자식노드)// 하급자
6. Leaf Node(Terminal Node): Child Node 가 하나도 없는 노드 // 막내
7. Sibiling(Brother Node): 형제 노드 , 동일한 Parent Node를 가진 노드 // 동기




이진트리 와 이진 탐색 트리 (Binary Search Tree)
이진트리 : 노드의 최대 Branch 가 2인 트리
이진 탐색트리 (BST):이진 트리에  왼쪽 노드는 해당 노드보다 작은 값, 오른쪽 노드에는 해당노드보다 큰값을 배치하는 조건이 있는 트리

주요용도 : 데이터 검색(탐색)
장점 : 탐색 속도를 개선할 수있음
단점 :

이진트리와 정렬된 배열간의 탐색 비교
